# Ruby Project Template


## initialize

```
$ wget https://github.com/kasei-san/ruby-project-template/archive/master.zip
$ unzip master.zip
$ mv ruby-project-template-master/ #{project name}
$ cd #{project name}
```

## Guard

```
$ bundle exec guard
```
